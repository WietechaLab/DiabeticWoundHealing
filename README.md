# DiabeticWoundHealing
Code to reproduce the analyses and figures in the diabetic wound healing study of Wietecha, et al. 2026

The code is split up into R scripts and annotated according to the figure numbers in manuscript:
https://www.biorxiv.org/content/10.64898/2026.04.26.720829v1.full

The RDS files of Seurat objects needed to reproduce the results can be found on Zenodo at the following DOI:
10.5281/zenodo.20549537

The single-cell sequencing datasets of mouse db/db and ND wounds are uploaded to NCBI Sequence Read Archive (SRA) with BioProject number PRJNA1467292 and to the Gene Expression Omnibus (GEO) with accession number GSE333591 (Reviewer token: kdafmyoknzappof)
